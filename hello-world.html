Hello World！

Date 周一 11 八月 2014 Tags 杂物 / 随笔 / python

断断续续折腾了好几天，终于是吧这个博客弄起来了。

用pelican配的github pages，不得不说python2在windows下的编码问题真是令人蛋碎啊。从安装开始就不停的报unicodedecodeError。差点把我搞疯了。这是逼我转投python3的节奏么。

不管怎样，问题终究是解决了。解法是在每一个报错的文件里加上

import sys
reload(sys)
sys.setdefaultencoding(sys.getfilesystemencoding())

不小心误入这里的朋友也许可以用上。

最后感谢一下frantic1048提供的pelican使用教程。
